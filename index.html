Introduction to Econometrics

as taught at UCSB

  • Econometrics is the application of statistics, mathematics, and computer science, to economic data, to answer questions like these:
  • 1. How do we test a given scientific hypothesis?
  • 2. How do we measure the effect of exogenous or endogenous variables in a model?
  • 3. How do we forecast the future?
  • The main tool you will learn in this course is multiple regression.
